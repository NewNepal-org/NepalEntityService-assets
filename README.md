# NepalEntityService-assets

Nepal Entity Service assets repository with Jekyll website for GitHub Pages.

## Development

```bash
# Install dependencies
bundle install --path vendor/bundle

# Serve locally
bundle exec jekyll serve
```

## Assets

Store assets in the `assets/` directory:
- `assets/images/` - Image files

## Deployment

Push to GitHub and enable Pages in repository settings. Site will be available at:
`https://yourusername.github.io/NepalEntityService-assets`
